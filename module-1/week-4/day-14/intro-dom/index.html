D.O.M (A.P.I)

Document Object Model : Application Programming Interface.
Une API c'est un ensemble de fonctions/propriétés qui sont là, disponibles pour coder/développer des applications.
Le DOM : le modèle object du document : c'est un ensemble de propriétés et de fonctions présentent pour vous permettre de coder des application en relation avec n'importe quel document HTML/XML.
- Hypertext Markup Langage - XML : Extended Markup Langage (Extended : la possibilité de créer des propres balises et attributs) HTML5 a remplacé ces fonctionnalités. Et J.S.O.N (Javascript Object Notation) a pris le dessus en ce qui concerne la structuration des données et leur échanges entre systèmes d'information (entre serveurs ...).
